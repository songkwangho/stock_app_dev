## 1. 한국 주식 시장 기본 구조

### 거래소
- **KOSPI** (한국거래소 유가증권시장): 삼성전자, 현대차 등 대형주. 종목코드 6자리(예: 005930)
- **KOSDAQ** (코스닥): 중소·기술주 중심. 종목코드 6자리(예: 035720)
- **거래 시간**: 평일 09:00~15:30 KST. 공휴일 휴장.
- **가격 제한폭**: 전일 종가 대비 ±30% (상한가/하한가)

### 주요 지표 의미와 초보자 설명
| 지표 | 계산식 | 초보자 설명 | 낮으면 | 높으면 |
|------|--------|------------|--------|--------|
| PER | 주가 / EPS | 지금 주가가 1년 이익의 몇 배인가 | 저렴할 수 있음 | 비쌀 수 있음 |
| PBR | 주가 / BPS | 장부가치 대비 주가 배수 | 자산 대비 저평가 | 자산 대비 고평가 |
| ROE | 당기순이익 / 자기자본 × 100 | 투자 대비 얼마나 벌었나 | 효율 낮음 | 효율 높음 (15%+ 우량) |
| EPS | 당기순이익 / 발행주식수 | 주당 순이익 | - | - |
| PEG | PER / EPS성장률 | PER을 성장률로 보정 | 성장 대비 저평가 | 성장 대비 고평가 |

> **초보자 주의**: PER 비교는 반드시 **같은 업종끼리** 해야 합니다. IT 기업과 은행을 PER로 비교하면 의미 없음.

### 적정 지표 범위 (업종별 편차 있음, 참고용)
- PER: 10~20배 = 적정, 20배+ = 고평가, 10배 미만 = 저평가 가능성
- PBR: 1 미만 = 자산 대비 저평가, 3+ = 자산 대비 고평가
- ROE: 5% 미만 = 낮음, 10~15% = 양호, 15%+ = 우량

## 2. 기술적 분석 지표 도메인 지식

### RSI (Relative Strength Index, 상대강도지수)
- **계산**: 14일 평균 상승폭 / (평균 상승폭 + 평균 하락폭) × 100
- **범위**: 0~100
- **해석**:
  - 70 이상: 과매수 (단기 조정 가능성)
  - 30 이하: 과매도 (반등 가능성)
  - 30~50: 하락 추세에서 회복 중인 신호 (이 앱의 보정 구간)
- **초보자 설명**: "주가가 얼마나 빠르게 오르고 있는지 측정해요. 너무 빨리 오르면 쉬어갈 수 있어요."
- **한계**: RSI가 높다고 반드시 떨어지는 건 아님. 추세가 강하면 70 이상 지속 가능.

### MACD (Moving Average Convergence Divergence)
- **계산**: EMA(12) - EMA(26), 시그널선 = EMA(9)
- **히스토그램**: MACD - 시그널선
- **해석**:
  - 히스토그램 > 0 & 증가: 매수 모멘텀 강화
  - 히스토그램 < 0 & 감소: 매도 압력 강화
  - 골든크로스(MACD가 시그널선 상향 돌파): 매수 신호
  - 데드크로스(MACD가 시그널선 하향 돌파): 매도 신호
- **초보자 설명**: "단기와 장기 평균 가격의 차이로 방향을 봐요. 차이가 커지면 그 방향 추세가 강한 거예요."
- **한계**: 후행 지표이므로 추세 전환 후 신호가 나옴.

### 볼린저밴드 (Bollinger Bands)
- **계산**: 20일 이동평균 ± 2 표준편차
- **%B**: (현재가 - 하단밴드) / (상단밴드 - 하단밴드) × 100
- **해석**:
  - %B < 20: 하단 근접 (과매도, 반등 가능)
  - %B > 80: 상단 근접 (과매수, 조정 가능)
  - 밴드 폭이 좁아짐: 변동성 수축, 큰 움직임 예고
- **초보자 설명**: "주가가 평소 움직이는 범위를 표시해요. 범위를 벗어나면 특이한 상황이에요."
- **한계**: 추세 장세에서는 상단밴드를 따라 지속 상승 가능.

### 이동평균선 (SMA: Simple Moving Average)
- **5일선**: 단기 추세 (1주일). 가장 민감.
- **20일선**: 중기 추세 (1달). 중요 지지/저항선.
- **정배열**: 주가 > 5일선 > 20일선 → 상승 추세
- **역배열**: 주가 < 5일선 < 20일선 → 하락 추세
- **초보자 설명**: "최근 N일 동안의 평균 가격이에요. 주가가 평균 위에 있으면 최근 상승 중인 거예요."

## 3. 수급 (투자자별 매매동향) 해석

### 투자자 유형
| 투자자 | 특징 | 초보자 설명 |
|--------|------|------------|
| 외국인 | 글로벌 기관투자자, 정보력 높음 | 해외 큰손들. 이들이 사면 긍정 신호로 보는 경우 많음 |
| 기관 | 증권사, 보험사, 펀드 등 | 국내 전문 투자 회사들 |
| 개인 | 일반 투자자 | 일반인 투자자. 역발상 지표로 쓰이기도 함 |

### 수급 해석 주의사항
- 외국인/기관 순매수 = 반드시 오른다는 의미가 아님
- 단기 매매 vs 장기 포지션 구분 어려움
- 수급 단독보다 기술지표·밸류에이션과 함께 해석 필요
- **이 앱의 수급 스코어**: 외국인(가중 1.2) + 기관(가중 0.8), 10일 감쇠 방식

## 4. HoldingOpinion 로직의 현실적 의미

현재 구현된 5단계 판단을 실제 투자 맥락에서 해석:

### 각 판단의 의미
| 판단 | 표시 라벨 | 시나리오 | 권고 행동 |
|------|----------|---------|---------|
| 손절(-7%) | 주의 필요 | 평단가 대비 7% 이상 하락 | 추가 손실 전 대응 검토 |
| 이중 이탈 | 주의 필요 | 5일선·20일선 모두 이탈 | 추세 반전 가능성 고려 |
| 단기이탈+중기지지 | 관망 | 5일선 아래지만 20일선 위 | 20일선 지지 여부 확인 후 판단 |
| 5일선 근접 | 추가 검토 | 5일선 ±1% 이내 | 지지 확인 후 매수 검토 가능 |
| 정배열 | 보유 | 상승 추세 유지 중 | 현 상태 유지 |

### 손절 기준 -7%의 근거
- 한국 주식 일일 등락폭 ±30% 제한 내에서 7%는 유의미한 손실
- 단기 변동성 범위(통상 5~10%)의 하단
- **이 앱의 면책**: 이 기준은 알고리즘 참고값. 개인별 리스크 허용 범위에 따라 다름

## 5. MarketOpinion 스코어링 현실성 검토

### 10점 스코어링의 한계 (반드시 UI에 표시)
1. **백테스팅 미검증**: 7점=긍정 임계값은 데이터 기반이 아닌 임의 설정
2. **시장 환경 미반영**: 강세장/약세장에 따라 같은 점수의 의미가 다름
3. **종목 특수성 미반영**: 바이오 주식은 PER 무의미, 금융주는 PBR 중요
4. **단기 지표 편중**: RSI/MACD는 단기 신호, 장기 투자 판단과 다를 수 있음

### 섹터별 지표 해석 차이 (스크리너 사용 시 중요)
| 섹터 | PER 기준 | PBR 기준 | 주요 지표 |
|------|---------|---------|---------|
| 기술/IT | 20~40배 정상 | 3~10배 | ROE, 매출 성장률 |
| 금융/지주 | 5~15배 | 0.5~1.5배 | PBR, 배당수익률 |
| 바이오 | 적자 많음 (의미 없음) | 2~5배 | 파이프라인, 임상 단계 |
| 에너지/소재 | 10~20배 | 0.8~2배 | EBITDA, 원자재 가격 |
| 자동차 | 8~15배 | 0.5~1.5배 | 영업이익률, 재고회전율 |

## 6. 추천 기능 현실성 평가

### 현재 추천 방식의 한계
- **수동 추천 20개**: 편집 시점의 분석가 의견 반영. 업데이트 없으면 낡음
- **알고리즘 추천 (market_opinion='긍정적')**: 순수 정량 지표만 반영, 기업 내러티브 미반영
- **적정가(fair_price) 신뢰도**: 애널리스트 목표가는 6~12개월 기준, 단기 가격과 다름

### 사용자에게 설명해야 할 것
- 추천 = 매수 신호가 아님
- 한 시점의 스냅샷, 기업 상황은 계속 변함
- 분산 투자가 중요함 (한 종목에 집중 투자 위험)

### 투자 권고로 해석될 수 있는 표현 → 중립 표현 변환
| 위험 표현 | 안전 표현 |
|----------|---------|
| 매도 신호 | 가격 하락 경고 |
| 매수 타이밍 | 가격 지지 알림 |
| 추가매수 | 추가 검토 |
| 저평가 | 저평가 분석 결과 |

### 면책 문구 원칙
1. "투자 결정은 항상 본인이 직접 하세요" — 책임 전가 방지
2. "이 앱은 정보 제공 도구입니다" — 투자자문업 아님을 명시
3. "과거 성과가 미래 수익을 보장하지 않습니다" — 알고리즘 한계 명시
4. "실제 거래는 증권사 앱에서 직접 진행해 주세요" — 거래 기능 없음 명시

## 9. 포트폴리오 관리 UX 원칙

### 초보자가 자주 혼동하는 것
1. **평단가 vs 현재가**: "평단가"(내가 산 가격)와 "현재가"(지금 시장 가격)의 차이
2. **수익률 계산**: (현재가 - 평단가) / 평단가 × 100
3. **포트폴리오 비중**: 전체 투자금 중 이 종목이 차지하는 %
4. **평가금액**: 현재가 × 보유 수량

### 입력 폼 UX 가이드
```
평단가 입력 레이블: "내가 산 가격 (평균)"
→ 부연: "여러 번 나눠 샀다면 평균값을 입력해요"

수량 입력 레이블: "보유 주식 수"
→ 부연: "증권사 앱에서 확인할 수 있어요"

수익률 표시: "+5.2% (+1,040,000원)"
→ 퍼센트와 절대금액 모두 표시
```

## 10. 한국 주식 앱 경쟁 환경 이해

### 주요 경쟁/유사 앱
| 앱 | 주요 기능 | 차별점 |
|-----|---------|-------|
| 한국투자증권 | 실거래 + 분석 | 거래 가능 |
| 키움증권 | 실거래 + 차트 | 전문가 향 |
| 토스증권 | 실거래 + 쉬운 UI | 초보자 친화 |
| 네이버 증권 | 정보 제공 | 뉴스·지표 중심 |
| **이 앱** | 포트폴리오 관리 + 분석 참고 | 거래 없음, 초보자 설명 중심 |

### 이 앱의 포지셔닝
- "내 주식을 한 곳에서 관리하고, 쉬운 언어로 분석 결과를 보여주는 도구"
- 실거래 미지원 → 장점: 규제 부담 없음, 단점: 사용자 이탈 (거래하러 다른 앱 이동)
- 핵심 가치: 초보자도 이해할 수 있는 분석 설명
